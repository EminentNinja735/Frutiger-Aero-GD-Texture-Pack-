# Install instructions
1. Dowload Geode if you don't already have it.

2. Dowload the Happy Textures Mod, and the Texture Loader Mod.

3. Dowload whichever Texture Pack you want to.

4. then go to your files
_________________________________

# android version
5. Go to Cx File Explorer (If you don't have it download it from the app store)

6. Copy the zip

7. click main storage

8. Click Android

9. Click Media

10. Click Geode

11. Click "config"

12. press "geode.texture-loader"

13. press "packs"

14. Put the zip file into the packs folder
________________________

# Laptop/Pc Version
5. Copy The Zip

6. Go To Steam

7. Click Browse Local Files

8. Click Geode
__________________________________________________
Everything else after 8. is the same as android.
